Bounded \(\chi^2\) Implies Contiguity

Theorem

We say that \(P_n\) is contiguous with respect to \(Q_n\) if for any \(E_n\) such that \(Q_n(E_n) \rightarrow 0\) we have \(P_n(E_n) \rightarrow 0\). Suppose that \(\textbf{C} := \sup_n \chi^2(P_n\|Q_n) < \infty\), then \(P_n\) is contiguous with respect to \(Q_n\).

Proof

Suppose that \(Q_n(E_n) \rightarrow 0\). Then \[ P_n(E_n) = \mathbb{E}_{P_n} \text{1}_{E_n} = \mathbb{E}_{Q_n} \text{1}_{E_n} \dfrac{dP_n}{dQ_n} \] \[ \leq \sqrt{ \mathbb{E_n}_{Q_n} \left ( \dfrac{dP_n}{dQ_n} \right)^2 Q_n(E_n) } \leq \sqrt{ ( \chi^2(P_n\|Q_n) +1 ) Q_n(E) } \] \[ \leq \sqrt{ (\textbf{C} + 1) Q_n(E_n)} \rightarrow 0 .\]
