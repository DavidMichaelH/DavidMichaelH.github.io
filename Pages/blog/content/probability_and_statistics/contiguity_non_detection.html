Contiguity implies impossibility of detection

Theorem

We say that \(P_n\) is contiguous with respect to \(Q_n\) if for any \(E_n\) such that \(Q_n(E_n) \rightarrow 0\) we have \(P_n(E_n) \rightarrow 0\). Then suppose that \(P_n\) is contiguous with respect to \(Q_n\). Let \(\phi\) be a classifier for the hypothesis testing problem given by $$ H_0 : X \sim P_n $$ $$ H_1 : X \sim Q_n .$$ Then the testing risk \[ \mathcal{R}_n(\phi) = P_n(\phi = 1) + Q_n(\phi = 0) \] cannot tend to zero.

Proof

Letting \(E_n = \{\phi = 0\}= \{\phi = 1\}^c \). Thus if \(Q_n(\phi = 0) \rightarrow 0\), then \(P_n(\phi = 1) \rightarrow 1\). Therefore \(\mathcal{R}_n(\phi) \not\rightarrow 0\).
